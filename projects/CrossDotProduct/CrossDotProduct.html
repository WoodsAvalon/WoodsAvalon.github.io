Cross and Dot
Product Calculator

GitHub

Calculates the cross and dot product of two vectors supplied to it.

This project was an attempt to do basic mathematics within Java as well as import in a custom library, currently listed as the project Custom Library. While the original idea was a GUI based program that could target input and output files to do calculations from, it was later decided to be a waste of time for this program. In the future, calculating the Cross and Dot Products will be used in multple programs, but this one is just a simple demonstration of it calculating it.

To run this program download the program and open the folder in the command line interface. Run the program using the command java -jar CrossDotProduct.jar. Once the program is running, it will request two vectors in the command line interface. Each vector is intered in the x,y,z format. Once both vectors are entered, it will calculate and display the cross and dot products and then end the program.
