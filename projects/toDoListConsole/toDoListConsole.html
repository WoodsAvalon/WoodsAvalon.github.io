The second attempt of the console version of the To Do List.

This project was original started as a means of rebuilding my programming skills in Java. Java itself was chosen due to the ability to quickly switch between Windows and Linux with a quick and easy means of adding in a GUI at a later date. This ended up as the second attempt of the project due to the first one getting deleted from my hard drive by accident.

To run this program download the program and open the folder in the command line interface. Run the program using the command java toDoListConsole.toDoListConsole in the bin folder. No arguments are required until the program is running. Once it is running, use a or A to add a task, c or C to complete the task, d or D to delete the task and e or E to exit the program. On load, it will attempt to load from a plain text file any already created task and on exit, it will attempt to save the current task list into the same plain text file.
