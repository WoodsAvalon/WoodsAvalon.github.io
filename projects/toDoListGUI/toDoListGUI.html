To Do List
GUI Version

GitHub

Modifing the console to do list to have a Swing based graphical user interface.

This project revolved around using Swing to make a functional GUI in Java. While it used code from the first attempt of the console version of the To Do List, it was altered significantly to create the desired appearance.

To run this program download the program and open the folder in the command line interface. Run the program using the command java toDoList.toDoListMain in the bin folder. No arguments are required to run the program. Once the program is running, you can add a new task by clicking the add task button. It will create a blank task with the current time as the start time. Then click on Add Text Here and enter the text for the task and press enter or return when done. To complete a task, click on the task, turning the pending label background yellow, then click the complete task button. It will set the current time as the completed time. To delete a task, click on the task, turning the pending label background yellow, then click the delete task button. It will remove the task from the list. To exit the program, either click on the X in the upper right corner of the window or on the exit button. When the program shuts down, it will automatically save the task list to a plain text file and when loading, will attempt to read the same file, loading recorded task into the program.
